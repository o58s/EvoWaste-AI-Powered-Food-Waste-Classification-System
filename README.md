EvoWaste is a full-stack AI application designed to help users classify and track their food waste using real-time image recognition. The system uses a YOLO deep learning model trained on a custom dataset I created using Roboflow, allowing accurate classification of waste into four categories: Edible, Animal Feed, Compostable, and Non-Reusable.

During development, I debugged complex backend logic, file handling, and model predictions to ensure smooth functionality and accurate results.
